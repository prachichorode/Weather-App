# Weather-App
# 🌦️ Weather App

A modern and responsive Weather Application built using HTML, CSS, and JavaScript.  
This app provides real-time weather updates, hourly forecasts, AQI details, and dynamic weather backgrounds using the OpenWeather API.

---

## 🚀 Features

- 🌍 Search weather by city name
- 📍 Current location weather detection
- 🌡️ Real-time temperature updates
- 📅 5-Day weather forecast
- 🕒 Hourly forecast section
- 🌬️ Air Quality Index (AQI)
- 🌅 Sunrise & Sunset timings
- 💧 Humidity, Pressure, Wind Speed, Visibility
- 📊 Hourly temperature chart
- 🌙 Dark / ☀️ Light Mode
- 🎨 Dynamic background based on weather condition
- 📱 Fully responsive design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- OpenWeather API
- Font Awesome
- Moment.js
- Chart.js
